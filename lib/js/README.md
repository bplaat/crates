# WIP JavaScript Interpreter

A WIP JavaScript interpreter

## Examples

-   operators
-   tenary
-   strings
-   if, while, do while, for,
-   switch
-   typeof
-   var
-   function
-   array
-   object
-   class

```js
10 + 20;
10 * (34 - 12);
8 ** (2 * 2);
a = 10;
a * 8;
b = 90;
a * b;
20, 30, 40;
(a = b = 20), a + b;
```
